# Story Arc Diagrams

This directory contains visual diagrams and state machines that map out the architecture and progression of the Prose Ada story.

## Files

- **story-arc-state-machine.md** - Comprehensive state machine diagram showing:
  - Book-by-book progression
  - Key states and transitions
  - Timeline convergence mechanics
  - Character arc progressions
  - Structural evolution (Single Timeline → Bundle → Superbundle → Lattice)
  - Narrative pacing visualization

- **story-timeline-visual.md** - Timeline visualizations including:
  - Gantt chart showing story progression
  - Timeline convergence progression graph
  - Ada's life arc vs. story timeline
  - Sequence diagram of key events

- **view-diagrams.html** - **EASIEST WAY TO VIEW**: Open this file in any web browser to see all diagrams rendered visually!

## How to View the Diagrams

### 🎯 Easiest Method: HTML Viewer
**Double-click `view-diagrams.html`** in your file explorer, or right-click and select "Open with" → your web browser. This will show all diagrams rendered beautifully!

### Alternative Methods:

1. **VS Code / Cursor**:
   - Install the "Markdown Preview Mermaid Support" extension
   - Or install "Mermaid Preview" extension
   - Open the `.md` files and use the preview pane

2. **Online Editor**:
   - Go to [mermaid.live](https://mermaid.live)
   - Copy the code from inside the ` ```mermaid ` code blocks
   - Paste into the editor to see the diagram

3. **GitHub**:
   - If you push to GitHub, Mermaid diagrams render automatically in markdown files

## Diagram Types

### State Machine Diagrams
Visualize the story as a series of states and transitions, showing how the narrative flows from one key moment to the next.

### Timeline Convergence Mechanics
Shows how timelines evolve from individual realities to converged structures, and how Arboros functions as the underlying cosmic architecture.

### Character Arc Progressions
Tracks how main characters (Ada, Edgar Potter, Benny, Rubies) evolve throughout the story.

### Narrative Pacing Timeline
Illustrates how the story accelerates through time, with Books One-Two focused on Ada's lifetime, and later books spanning generations.

## Key Concepts Visualized

- **Consciousness Anchor**: How Ada becomes the pivot point for timeline convergence
- **Resonance Matching**: The mechanism by which timelines attract and merge
- **Structural Evolution**: The progression from single timelines to cosmic lattice
- **Arboros**: The self-creating cosmic meta-structure underlying all reality
