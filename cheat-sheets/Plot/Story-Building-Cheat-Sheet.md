# Ultimate Plot/Story Building Cheat Sheet

This cheat sheet distills invariant principles from enduring stories (e.g., Aristotle's Poetics, modern hits like Dune, The Godfather). Prioritizes mechanics over style: load-bearing elements that fail most drafts. Use as reference/checklist during outlining, drafting, revision. Relationships: Principles feed into patterns; patterns into outline; all tested via revision. Explicit rules minimize guesswork—apply sequentially.

## 1. Core Invariants (Must-Haves for Beloved Stories)
These recur in 90%+ of high-impact narratives. Absence causes drift/failure.

- **Living Promise**: Story commits to one primary emotional payoff early (by ~10% mark).  
  - Types: Wonder (discovery), Dread (tension), Romance (connection), Moral Pressure (dilemma), Puzzle (resolution), Catharsis (release).  
  - Rule: Escalate payoff cost progressively; unkept promise = reader betrayal.  
  - Check: "This story promises [specific payoff, e.g., 'thrill of outsmarting a crushing system']. Test first chapter summary—if no visceral hook, rewrite.  
  - Relation: Ties to escalation; unmet = flat plot.

- **Desire Under Constraint**: Protagonist pursues concrete want; blocked by scaled opposition.  
  - Components: Want (vector: direction + magnitude), Constraint (opposing force), Collision (forces plot).  
  - Rule: No free wins; constraint scales with want (low want + high constraint = whiny; inverse = boring). Spill "blood" (literal/metaphorical) per collision.  
  - Check: Map as equation: Plot = ∑(Want × Constraint) over acts. No pain = no story.  
  - Relation: Fuels character change; unresolved = tour, not arc.

- **Character Arc (Directional Change)**: Protagonist shifts belief/model of reality.  
  - Types: False Belief → Pressure → Reversal → New Belief; or Self-Protection → Risk → Commitment.  
  - Rule: Binary flip (no gray ambiguity for impact); static only for satire/absurdism.  
  - Check: "Page-1 belief: [false axiom, e.g., 'Power solves loneliness']. Page-end: [proven reversal]. Readers crave simulated transformation.  
  - Relation: Plot = machine to break belief; ties to theme as argument.

- **Inevitable Escalation**: Events chain causally from choices/flaws.  
  - Mechanism: Move → New Problem → Tougher Move → Climax (only option left).  
  - Rule: Link each to prior flaw/choice; random = cheap. "Of course!" hindsight.  
  - Check: Decision tree per act—if branch not flaw-derived, cut. 80% amateur failures here.  
  - Relation: Builds page-turner; earns emotion via irreversibility.

- **Earned Emotion**: Result of choices × costs × irreversibility, filtered by character.  
  - Rule: Quantify costs (e.g., dignity=5pts, betrayal=10pts); threshold per genre (e.g., 20pts drama). Stoic tear > crybaby sob.  
  - Check: No unearned (reviled like "Mary Sue"); write loss after commitment, not raw sadness.  
  - Relation: Culminates arc/escalation; unearned = gas leak smell.

## 2. Plot Design Rules (Operational Mechanics)
Cause-driven; prioritize hindsight logic over surprise.

- **Cause > Surprise**: Plant seeds early; harvest reverse-order.  
  - Rule: 3-5 causes per act; no causeless twists (collapse on re-read). 90% "clever" plots fail this.  
  - Relation: Enables inevitability; subverts tropes.

- **Stakes = Personal/Moral Cost**: Not world-ending; tie external to ego (e.g., betray friend, fail child-self).  
  - Rule: Scale inverse to scope (global needs micro-anchor). Irreparable loss / identity.  
  - Relation: Amplifies emotion; low stakes dilute arc.

- **Scene Triad**: Goal (immediate want), Conflict (block: person/system/self), Clock (urgency: ticking timer/erosion).  
  - Rule: Missing one = boredom; merge/delete scene. Clock decays if inaudible.  
  - Relation: Micro-version of escalation; builds rhythm.

- **Show Don't Tell (Mechanized)**: Externalize via action/dialogue/sensory.  
  - Rule: Convert 80% tells (e.g., "He was angry") to evidence (e.g., "Knuckles white, door slammed"). Exceptions: time jumps.  
  - Relation: Enables inference/simulation; ties to specificity.

## 3. Common Patterns (Spines to Build Around)
Pick one; mixing dilutes (stick to mastery). Failure modes explicit.

| Pattern | Best For | Structure | Failure Mode | Relation to Invariants |
|---------|----------|-----------|--------------|------------------------|
| Hero's Journey (Identity via Ordeal) | Epic/Mythic/Transformation | Ordinary World → Call → Refusal → Mentor → Threshold → Tests → Ordeal → Reward → Return Changed | Bloated if not flaw-linked | Arc as core; escalation in tests/ordeal |
| Pressure Cooker (Goal via Obstacles) | Thriller/Heist/Survival/Sports | Mission → Complications → Setback → Desperate Plan → Climax | Deus ex resolutions | Escalation engine; constraint heavy |
| Mystery (Truth via Investigation) | Detective/Conspiracy/Intellectual | Question → Clues → Misleads → Revelation → Recontextualize | Causeless reveals | Puzzle promise; cause > surprise |
| Relationship Arc (Connection via Vulnerability) | Romance/Buddy/Family | Distance → Friction → Intimacy Attempt → Rupture → Honest Choice → New Bond | Unearned intimacy | Emotional stakes; arc in commitment |
| Tragedy (Flaw via Inevitability) | Cautionary/Weighty | Strength-to-Flaw → Narrowing Choices → Self-Recognition (Late) | Avoidable flaw | Inevitable escalation; moral theme |

Rule: Match to theme; mismatch fractures story.

## 4. Philosophy (Underlying Logic)
Stories as simulators/arguments—explicit for theme distillation.

- **Meaning Simulators**: Safe run of "If I choose X, cost? Who become?"  
  - Rule: Model universal dilemma (e.g., ambition vs. humanity); no simulation = empty (e.g., plotless porn).  
  - Relation: Invariants provide simulation fidelity.

- **Argument via Events**: "When want X, tend Y; truth Z."  
  - Rule: Syllogism: Premise (want) → Major (block) → Conclusion (truth). Discover, don't preach; unexamined = accidental bias.  
  - Relation: Theme emerges from arc/escalation; preachy if inserted.

## 5. Craft Attributes (Execution Separators)
From fine to loved: Specificity, contrast, subtext.

- **Specificity (Anti-Generic)**: Unique objects/rituals/sensory/rules.  
  - Rule: Upgrade 70% nouns/adjs (e.g., "sword" → "rusted falchion with runes"). Sensory clusters: smell + texture + history.  
  - Relation: Grounds simulation; combats forgettable.

- **Contrast (Emotional Variance)**: Tension-relief, humor-horror, beauty-ugliness.  
  - Rule: Alternate valence every 3-5 scenes; waveform chart in outline. Monotone = fatigue post-50 pages.  
  - Relation: Sustains pacing; amplifies emotion.

- **Dialogue as Subtext**: Negotiation/status/avoidance/reveal (not info-dump).  
  - Rule: Layers: Surface + Intent + Slip. Strip dumps; if transactional only, cut.  
  - Relation: Shows character; earns subtext emotion.

- **Tropes as Tools**: Recycle + subvert for originality.  
  - Rule: List 5 genre tropes; invert 3 (e.g., early mentor death → solo flounder). Unchanged = noise.  
  - Relation: Subverts via cause/surprise; fits patterns.

## 6. Pacing & Structure (Rhythm Control)
Pace = info density / emotional variance.

- Rule: Chapters as sine waves: Build (20-30% setup, low action), Peak (50% conflict), Trough (20-30% reflection). Vary sentences: short=tension, long=immersion.  
- Check: Poor pacing kills > bad plots; skim if density flat.  
- Relation: Macro triad; prevents fatigue in escalation.

## 7. Outline Method (Minimal 7-Line Starter)
Iterative: Draft, reverse-engineer gaps.

1. Protagonist: [Trait + Wound].  
2. Want: [Concrete external goal].  
3. Need: [Resisted internal shift].  
4. Opponent/Force: [Cost-maker].  
5. Point of No Return: [Lock-in choice].  
6. Worst Loss: ["All lost" moment].  
7. Final Choice: [Climax sacrifice reveals arc].  

Rule: Build novel from these; iterate for invariants. Mismatch to pattern = rework.

## 8. Revision & Ending (Failure-Proofing)
95% greatness in rewrites (e.g., Hemingway: 39 ending drafts).

- **Revision Core**: Not afterthought—excavate story.  
  - Phases: Structure (chains), Character (flips), Language (specificity/contrast).  
  - Rule: Cut 30% darlings; beta for promise/arc; iterate until ironclad. First drafts suck universally.  
  - Relation: Tests all above; exposes suckage early.

- **Ending Integrity**: Resolve arc + theme, not just plot.  
  - Rule: Climax = choice proving change; Denouement = 5-10% new normal. "Of course!" + catharsis. Weak = erases goodwill (e.g., GoT backlash).  
  - Relation: Culminates invariants; rework from midpoint if fail.

## 9. Market/Timeless Balance (Optional Pragmatics)
Hard truth: Beloved = commercial + enduring; ignore audience = obscurity (<1k sales common).

- Rule: Timeless via universals; timely via hooks (trends/fears). Research comps; self-indulgent = niche.  
- Relation: Align promise/arc to reader simulation needs.
