# Chapters

Each chapter directory contains:
- Draft files (`.md`, `.txt`, `.docx`, etc.)
- Research materials specific to that chapter
- Inspiration images and mood boards
- Historical texts and references
- Character notes relevant to the chapter
- Plot notes and scene ideas

## Chapter Structure

Create a new directory for each chapter (e.g., `chapters/01-prologue/`, `chapters/02-chapter-name/`).

Each chapter directory can have:
- `draft.md` or `draft.txt` - The main chapter text
- `notes.md` - Chapter-specific notes
- `research/` - Research materials for this chapter
- `images/` - Visual inspiration
- `references/` - Historical texts, quotes, etc.
