# Chapter Template

This is a template showing how to organize a chapter directory.

## Files

- `draft.md` - Main chapter draft (or use `.txt`, `.docx`, etc.)
- `notes.md` - Chapter-specific notes, ideas, and thoughts

## Directories

- `research/` - Research materials specific to this chapter
- `images/` - Visual inspiration and mood images
- `references/` - Historical texts, quotes, and references

---

*Copy this directory structure when creating a new chapter, or adapt as needed.*
